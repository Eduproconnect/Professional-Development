How to Use These Materials

This professional development package is designed for a full-day workshop (6 hours) focused on building relationships with students as the foundation of effective teaching. The materials can be adapted for different timeframes and contexts.

For Facilitators:

  • Begin with the Session Outline to understand the flow and objectives
  • Review the Presentation Slides to familiarize yourself with the content
  • Print activity handouts for all participants
  • Prepare video clips from the recommended sources
  • Review the research materials for deeper background knowledge

For Participants:

  • The Activity Handouts guide you through interactive exercises
  • The Key Quotes Handout provides inspiration and discussion points
  • The Resource Guide offers materials for continued learning
  • The Monday Morning Plan helps you implement strategies immediately

For Follow-Up:

  • Use the Evaluation Form to gather feedback
  • Refer to the Resource Guide for ongoing professional development
  • Revisit the Key Themes and Principles to reinforce learning

Customization Options

These materials can be customized for different educational contexts: - For elementary teachers: Emphasize classroom community-building activities - For secondary teachers: Focus on strategies for connecting with adolescents - For administrators: Highlight system-level supports for relationship-building - For shorter sessions: Select specific activities based on priority needs

Technical Requirements

For optimal delivery of this professional development session: - Computer with presentation software - Projector and screen - Internet access for video clips - Audio capability - Chart paper and markers for activities - Printed handouts for all participants
